This website presents current weather conditions and a 7-day dew point forecast.

Dew Point

The dew point is the temperature at which dew will form as air is cooled. It's a better measure than humidity for how humid it feels as the value does not vary with the air temperature the way the relative humidity percentage does. A higher dew point indicates more moisture in the air and therefore more humid conditions. Read more on Wikipedia

Author and Technology

It was built by Wesley Moore with inspiration from dewpoint.xyz. The application is implemented in Rust using the Rocket web-framework. The code is published on GitHub.
